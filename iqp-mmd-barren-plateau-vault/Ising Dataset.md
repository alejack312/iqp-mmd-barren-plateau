# D2 — Ising-Like Synthetic

Structured binary data generated by Gibbs-sampling a sparse Ising model. Tests how data correlations interact with the gradient landscape.

## Target Distribution

$$
p(x) \propto \exp\!\left(-\beta \sum_{\langle i,j\rangle} J_{ij}\, s_i s_j\right), \quad s_i \in \{-1, +1\}
$$

where:

- $\beta$ is inverse temperature (default 1.0)
- $J_{ij} \sim \mathcal{N}(0, \sigma_J^2)$ with $\sigma_J = $ `coupling_std / sqrt(n)`
- Couplings are **sparse** — nonzero only on edges of the topology graph
- Spins are converted to binary via $x_i = (s_i + 1)/2$

## Topologies

### `grid_2d`

- $n$ must be a perfect square
- Edges: horizontal + vertical nearest neighbors on an $L \times L$ grid
- Matches the [[ZZ Lattice Family|2D lattice circuit family]], so the data graph and the model graph align

### `erdos_renyi`

- Same bounded-degree calibration as the [[Erdos-Renyi Family|ER circuit family]]
- Default average degree 2.0

## Sampler

Sequential Gibbs (Glauber dynamics):

```python
for sweep in range(burn_in_sweeps + n_samples * thinning):
    for site in range(n):
        local_field = sum(weight * spins[neighbor] for neighbor, weight in adjacency[site])
        prob_up = 1.0 / (1.0 + exp(-2 * beta * local_field))
        spins[site] = 1 if rng.random() < prob_up else -1
    if in sampling phase:
        record spins
```

- `burn_in_sweeps`: default `max(20n, 100)`
- `thinning`: default 2
- `num_chains`: default 4 (parallelized across chains by allocation)

## Config

```yaml
dataset:
  type: ising
  n_samples: 10000
  ising:
    beta: 1.0
    coupling_std: 1.0
    topology: grid_2d
    burn_in_sweeps: 320       # defaults to max(20n, 100)
    thinning: 2
    num_chains: 4
```

## Why Ising

From [[Project Overview]]:

- Structured pairwise correlations that the MMD² can actually see through weight-2 Z-words
- Closer to realistic physics data than Bernoulli baseline
- Lets you test whether the **data graph** aligning with the **model graph** changes scaling behavior (important for the `lattice` family)
